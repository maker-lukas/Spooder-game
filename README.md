## Spooder
[Play on itch.io](https://orangopontotango.itch.io/spooder)

A game about being a big spider in a procedurally generated world, 
made with custom coded procedural animations. Made in Godot.

*this project morphed from one idea into something a bit diffrent*

![Spider gameplay](/images/spooder_gifsicle.gif)

### Features
- **Procedural animation:** Spider movement and animations are generated entirely in code
- **Procedural world**: the world is also generated with only code and parameters can be changed in code
- **Enemies** Green orbs that attack down

### How to Play
- **WASD / Arrow keys** – Move the spider
- **Left / Right click** – Attack
- Survive blob hits before you're killed.
